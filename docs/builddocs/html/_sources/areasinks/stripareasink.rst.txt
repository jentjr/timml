Strip Area-Sink
------------------
The comprehensive potential for infiltration, :math:`N`, over a strip of width :math:`2b` is:
     
     .. math:: 
        
        \Theta=-\frac{1}{2}N(x^2 - b^2) + \Theta_0,\quad\quad -b \leq x \leq b
        
        \Theta=-Nb(x - b) + \Theta_0,\quad\quad b \leq x < \infty
        
        \Theta=Nb(x + b) + \Theta_0,\quad\quad -\infty < x \leq -b
